# 🎓 Student Management System

A simple, clean, and interactive web-based Student Management System built to manage student records efficiently.

## 🚀 Live Demo
Aap is project ko yahan live chala kar dekh sakte hain: 
👉 **[Live Project Link](https://remarkable-sherbet-074c69.netlify.app)**

## ✨ Features
* **Add Student:** Quickly add student details like Name, Roll Number, and Grade.
* **Dynamic Table:** View all added student details instantly in a clean tabular format.
* **Delete Record:** Easily remove any student record with a single click.
* **Fully Responsive:** Works perfectly on both desktops and mobile browsers.

## 🛠️ Tech Stack Used
* **HTML5:** For creating the structure of the web application.
* **CSS3:** For styling, layout, and clean user interface designs.
* **JavaScript (ES6):** For handling logic, array manipulation, and dynamic DOM updates.

## ⚙️ How to Run Locally
1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser (Chrome, Brave, Edge, etc.).
3. Boom! The application is ready to use.
